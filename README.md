# Class Education Website

This repository contains the static website for Class Education, an educational agency based in Azerbaijan.

## Overview

The site is built with simple static files:

- `index.html` for the main landing page
- Separate `.html` pages for announcements and university information
- Image assets used across the website

The project does not use any framework or build tool. It is designed to stay simple and easy to edit.

## Main Pages

- `index.html` - homepage
- `architecture-exam-italy.html` - architecture entrance exam announcement
- `azerbaijan-scholarship.html` - scholarship program page
- `sapienza.html` - Sapienza University page
- `siena.html` - University of Siena page
- `tor-vergata.html` - Tor Vergata University page
- `ulvi-private-card.html` - digital business card page

## Design

The website uses:

- HTML for structure
- Inline CSS for styling
- Small amounts of vanilla JavaScript for interactive parts like the announcements slider

## Deployment

The website is deployed manually on Netlify.

To publish updates:

1. Edit the HTML files locally.
2. Review the changes in a browser.
3. Upload the updated files to Netlify.
